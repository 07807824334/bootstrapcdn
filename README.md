# Bootstrap CDN

Node.js Port of [github.com/netdna/bootstap-cdn](https://github.com/netdna/bootstap-cdn).

### Requirements

1. [Node.js](http://nodejs.org/) (tested on `0.10.17`+).

### Running

##### Locally

```
make setup run
```

##### Demonized

```
# before start or after npm update
make setup

# start server
make start

# stop server
make stop

# restart server
make restart

# server status
make status

# view logs
tail -f ./logs/server.log
```
